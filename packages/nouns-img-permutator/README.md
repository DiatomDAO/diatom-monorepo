# nouns-img-permutator

Web app run on Firebase to test Noun permutations. 

## Hosting

### Build

```
npm install
```

### Run emulators

```
firebase emulators:start --only hosting
```

## Cloud functions

### Build

Change directory to `functions/` and run:

```
npm install
```

### Run emulators

```
firebase emulators:start --only functions
```
